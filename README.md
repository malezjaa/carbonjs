## Installation

**Requirements**

- [curl](https://curl.se/) - to install cargo (on linux and macos)
- [cargo](https://www.rust-lang.org/) - comes by default with rust

```bash
    1. Install rust.
    2. `cargo install CarbonJS` - case sensitive
```
